### 4.4 Sable adjudication

**Spec-first protocol caught microestrutura coverage** (latency + RLP) porque microestrutura é **artifact-domain** (Beckett + Nova authorities each cobrem). Specifications são consumed **verbatim** entre artifacts; contracts são byte-equal.

**Spec-first protocol failed metrics chain coverage** porque metrics chain é **cross-domain**:
- Mira owns metric formula (info_coef.ic_spearman) — spec correto.
- Beckett owns CPCV harness (per-trial path-PnL output).
- Mira owns reportagem (compute_full_report).
- Connection between Beckett harness output and Mira ReportConfig.ic_in_sample input is **nobody's clearly-documented authority** — orphan.
