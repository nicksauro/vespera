## Follow-ups (non-blocking)

1. **[MINOR] Exit 4 explicit wiring in thin CLI** — `scripts/run_materialize_with_ceiling.py` should `try: import psutil except ImportError: sys.exit(4)` at top-of-file to make ADR-2 exit-4 semantics deterministic rather than relying on Python's default ImportError rc=1. Follow-up story.

2. **[MINOR] `run_id` path-traversal hardening** — operator-supplied `--run-id` is used directly as filename fragment without sanitization. Currently acceptable (operator is trusted Gage). Recommend future hardening: `run_id` validated against `^[a-zA-Z0-9._-]+$` regex or sanitized via `pathlib.Path.name` check. Follow-up story.

3. **[MINOR] Schema docstring mismatch on `ts_brt`** — `telemetry_schema.py` L30 column comment says `"ISO-8601 with -03:00 offset, e.g. '2026-04-23T14:05:32-03:00'"` but `run_with_ceiling.py` `_now_brt_iso()` emits BRT-naive (no offset) via `datetime.now().isoformat(timespec="seconds")`. R2 invariant is BRT-naive throughout the codebase, so the implementation is correct; the schema docstring should be amended to `"ISO-8601 BRT-naive, seconds resolution"`. One-line docstring edit. Non-blocking, purely cosmetic.

4. **[Q-03/Q-04] `backfill_manifest.py` residual concerns** — remain open for T11 re-gate per story AC13; not in scope for #74/#75.

5. **[INFO] ADR-2 exit 3 cross-reference** — prior audit finding 8 (add footnote in ADR-2 exit-3 line pointing to v2 semantics) was addressed by Aria Finding 8 Option A — footnotes applied inline. Verified present at `memory-budget.md` L56.
