### LOW: 2 (informational; do NOT block T11 handoff)

1. **mypy `asdict` narrowing diagnostic** ‚Äî `scripts/run_cpcv_dry_run.py:629`
   ```python
   d = asdict(stamp) if dataclasses.is_dataclass(stamp) else dict(stamp)
   ```
   mypy 1.20.2 reports: `Argument 1 to "asdict" has incompatible type "DataclassInstance | type[DataclassInstance]"; expected "DataclassInstance"`. This is a known mypy limitation with `dataclasses.is_dataclass` not narrowing instance vs type. Runtime safe: callers pass `BacktestResult.determinism` (an instance, not a class). Suggested cosmetic fix (non-blocking): cast via `if isinstance(stamp, type): raise TypeError(...)` before the call, or use `# type: ignore[arg-type]` with a justification comment. Tracked for next housekeeping pass; does not affect AC12/AC13.

2. **Legacy 2026-04-25 telemetry artifact uses pre-superset schema** ‚Äî `data/baseline-run/cpcv-dryrun-T002-2026-04-25/telemetry.csv`
   This file was written by the original Beckett HALT report (`docs/audits/AUDIT-20260425-PRE-DRYRUN-CPCV.md`) BEFORE T002.0f's `MemoryPoller` existed; its columns are the diagnostic `(timestamp_brt, phase, fold_idx, available_mib, available_gib, total_mib, peak_commit_mib, peak_commit_gib, soft_halt_threshold_gib, note)` schema, NOT the Pax gap #2 superset that the new MemoryPoller writes. T002.0f code emits the canonical superset (verified by `test_telemetry_csv_columns_superset`). No action required ‚Äî kept as historical evidence of the pre-T002.0f HALT.
