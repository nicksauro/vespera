### §2.1 `enabled_for_phase: ["F"]` gating — confirmed

Beckett confirms: synthetic walk's `_walk_session_path` synthesizes prices on-demand at decision time → `mid_at_decision == mid_at_fill_after_latency` by construction → `slippage_latency_pts = 0` trivially. ### §2.2 Slippage formula composition — no double-counting confirmed

```
total_slippage_pts = roll_spread_half_points          # immediate spread cushion (existing)
                   + slippage_extra_ticks × tick_size # static extra-ticks override (existing)
                   + slippage_latency_pts             # NEW Phase F: temporal mid drift during round-trip
```

Three components are **temporally orthogonal**:
- `roll_spread_half_points` is the **half-spread** at the fill instant → spatial cost (cross the book once).
- `slippage_extra_ticks × tick_size` is a **static** override / safety margin → constant-tick conservative add.
- `slippage_latency_pts = sign × (mid_at_decision − mid_at_fill_after_latency)` is the **mid-drift during latency window** → temporal cost.

No two of these terms measure the same physical phenomenon. **No double-counting.** Engine-config v1.0.0 §slippage_model preserved verbatim; v1.1.0 ADDS `latency_model:` block (Beckett spec §4); cosign re-locked at Beckett N7+ smoke run.

### §2.3 Determinism — seed anchoring confirmed

## §3 Toy benchmark real-tape proposal — Beckett implementer's plan

Mira §8(f) mandates re-running Bailey-LdP 2014 toy benchmark on real-tape harness with seeds `[42, 1337, 271828, 314159, 161803]` UNMOVABLE, asserting **Δ DSR > 0.10 across 5 seeds preserved**. Beckett-as-implementer proposes:

### §3.1 Strategy A no-edge / Strategy B small-edge over real tape

Per Aria C-A6 (structural sibling, NOT fork): import the same fixture pattern from `tests/cpcv_harness/test_toy_benchmark_bailey_lopez_de_prado_2014.py` which encodes Bailey-LdP 2014 §3 + Table 2 toy benchmark. Strategy contracts:

| Strategy | Synthetic version (Phase E) | Real-tape version (Phase F) |
|---|---|---|
| **A — no-edge** | Random labels uniform on `{−1, 0, +1}`, seed-driven | Random entry direction `{long, short}` chosen via `numpy.random.default_rng(seed_external_to_tape)` at session-day seed; entries fired at the standard 4 windows (16:55/17:10/17:25/17:40 BRT); barriers triple-barrier per Mira §2.1 over the **real WDO tape**. By construction, expected edge = 0 because direction is random; non-zero variance because real tape has volatility. |
| **B — small-edge** | Synthetic noise + small +0.1σ drift for true label sign | Real WDO tape + tiny edge injection: bias direction probability 55%/45% in favor of the empirically-observed end-of-day fade direction (negative sign on `intraday_flow_magnitude > P-upper`, positive when < P-lower per parent thesis) → injects ~0.10 standardized edge per Bailey-LdP 2014 Table 2. Barriers triple-barrier real tape. |

**Discriminator power claim:** Δ DSR = DSR(Strategy B) − DSR(Strategy A) > 0.10 across all 5 seeds, replicating the Bailey-LdP harness witness on real tape source.

### §3.2 Why this preserves the harness witness

The harness measures **DSR sensitivity to label-sign signal-to-noise**. Mathematically, Strategy A's PnL distribution is symmetric around 0 (random direction × realized return); Strategy B's is shifted right by ~0.10σ × realized-return-volatility. Bailey-LdP 2014 §3 shows DSR distinguishes the two with statistical power ≥ 0.50 at sample size ≥ 30-50 events per trial (matches Mira §6 mandate).

**Real-tape consideration:** real WDO afternoon tape has time-varying volatility (heteroscedasticity). The harness must be robust to this — but the theoretical argument holds because Δ DSR depends on the difference of two means over the **same** noise realization, and heteroscedasticity affects both equally. Empirically: if Δ DSR collapses below 0.10 on real-tape, the **harness is broken**, not the strategy → Mira §7 Step 2 routes to `data_quality` bucket. This is by design.

Beckett accepts. These are post-spec-finalize Mira authority (Mira §8(f) verbatim). Three observations:
1. The 5 seeds span deterministic-different starting points; Bailey-LdP 2014 §3 pseudo-replicate convention.
2. NumPy `default_rng(seed)` with these seeds produces independent draws across the 5 fixture instances.
3. Beckett N7+ harness run will execute the toy benchmark **before** the strategy benchmark — a "smoke test on the harness" preceding the "smoke test of the strategy." Pass/fail of toy benchmark is GATE for proceeding to Strategy benchmark per Mira §7 Step 2.
